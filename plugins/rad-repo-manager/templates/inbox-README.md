# docs/inbox/ — staging for unfiled docs

Newly generated docs that aren't part of the active core or the reference catalog
land here: side brainstorms, smoke-test reports, AI-produced analyses, one-off notes.

This is a tray, not a home. Files here are **awaiting filing**, not authority.

- Agents: write any new `.md` that doesn't have a home into this folder.
- `/rad-repo-manager:wrapup` reports how many docs are sitting here.
- `/rad-repo-manager:analyze` files each one (fold / extract / promote / archive)
  with your confirmation.

Don't let this fill up — an overflowing inbox is the drift this repo is built to catch.
