# docs/archive/ — historical context only

Superseded plans, old handoffs, finished smoke notes, retired brainstorms, and
detailed/old decision rationale live here. **Not current authority.**

Agents do **not** read this folder during normal work — only when the user
explicitly asks for historical context.

Each archived file should carry a banner at the top so no one mistakes it for live:

```
> **Archived <YYYY-MM-DD>.** Historical context only — not current authority.
> Superseded by <current home, e.g. docs/prd.md>.
```

Suggested subfolders as volume grows: `old-plans/`, `handoffs/`, `smoke-notes/`,
`brainstorms/`, `decisions/`.
